# Structure of Sensors and Actuators
## Sensors and Actuators  
The <font style="color:rgb(0, 0, 0);">Coding Kit for BBC micro: bit</font> is equipped with a variety of sensors and actuators. These components are structurally classified into two main categories: single-sided pinhole connectors and double-sided pinhole connectors, offering users versatile connection options. This document provides an overview of the structural features of common sensors and actuators in the kit.  

For detailed performance specifications and functional applications of each sensor and actuator, please refer to the subsequent functionality introduction section.
